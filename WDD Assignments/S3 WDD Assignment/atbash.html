Atbash Cipher

The Atbash cipher is another ancient substitution cipher, dating back to around 500-600 BCE. It works by substituting the first letter of the alphabet (A) with the last letter (Z), the second letter (B) with the second-to-last letter (Y), and so on, following this pattern:

Plain: ABCDEFGHIJKLMNOPQRSTUVWXYZ

Cipher: ZYXWVUTSRQPONMLKJIHGFEDCBA

So, for example, the word "HELLO" would be encrypted as "SVOOL" using the Atbash cipher.

Unlike the Caesar cipher, which shifts letters by a fixed number of positions, the Atbash cipher is a more fixed substitution where each letter is replaced by its mirrored opposite in the alphabet.
